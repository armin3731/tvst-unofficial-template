
# TVST Template

An unofficial LaTeX template for Translational Vision Science & Technology (TVST) manuscript submissions.

## Features

- Structured abstract format (Purpose, Methods, Results, Conclusions, Translational Relevance)
- Two-column layout for main content
- Support for figures, tables, and subsections
- Author affiliations and correspondence information
- Mostly compliance with TVST submission guidelines (double check them yourself)

## Usage

1. Clone or download this repository
2. Edit `main.tex` with your manuscript content
3. Replace placeholder text and images as needed
4. Compile with your LaTeX editor
